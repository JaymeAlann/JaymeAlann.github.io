BPM Beats

Android Application that connects a users heartrate monitor to their music allowing for a more connected workout. Once the application is authenticated with the users Spotify Account, the application will prompt you to connect to a BLE bluetooth device. This limits the devices that a user can use to heartrate monitors. Once connnects the user selects the playlist they would like to listen to, a starting song, and the application handles the rest.

A few seconds before the song changes, the app will read the users current heartrate and input that into the spotify application to get a song from the playlist whoes tempo is closest to that of the user.
